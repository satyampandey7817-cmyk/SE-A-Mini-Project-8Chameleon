# Smart Student Presence System

A modern, real-time student attendance tracking and management platform. Built with Flutter and Firebase, this application features dedicated, secure, and role-based dashboards to streamline the classroom experience.

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (v3.0.0 or higher)
- Dart SDK
- Firebase CLI
- Active internet connection

### Setup Instructions
1. Clone the repository
2. Navigate to the project directory
3. Run `flutter pub get` to install dependencies
4. Configure Firebase credentials
5. Run `flutter run` for mobile/web
